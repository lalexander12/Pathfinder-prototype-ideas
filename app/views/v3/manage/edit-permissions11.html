Help with access types

Read profile only

Can search Pathfinder and read:

  • nominal profile information
  • caseload filters
  • referral records (pre-adoption)
  • caseloads
  • how release dates are worked out
  • guidance
  • sentence and release date

Read all, no update

The user has the same access as ‘read profile only’ and can also:

  • download CSV files
  • print caseload
  • get notifications

The user can also read:

  • offence history
  • staff caseloads
  • pre-sentence nominals (Formerly known as missing nominals)
  • nominal TACT status
  • caseload filters
  • referral records (pre adoption)
  • status and reason for inclusion (post adoption)
  • ideology
  • meeting notes
  • intelligence reports
  • actions, control and interventions
  • case notes
  • case timeline
  • CIM assessment (Controls Intervention Matrix)
  • uploaded documents (for example, ERG and PGF)
  • link to DPS profiles (prison users only)
  • categorisation and re-categorisation status (prison users only)

Read all, limited update

The user has the same access as ‘read all, no update’ and can also:

  • upload documents (for example, ERG and PGF)
  • add case notes
  • change staff caseloads
  • change caseload filters
  • delete documents owned by user

Full use

The user has the same access as ‘Read all, limited update’ and can also edit:

  • staff caseloads
  • pre-sentence nominals (Formerly known as missing nominals)
  • nominal TACT status
  • caseload filters
  • referral records (pre adoption)
  • status and reason for inclusion (post adoption)
  • ideology
  • meeting notes
  • intelligence reports
  • actions, control and interventions
  • case notes
  • case timeline
  • CIM assessment (Controls Intervention Matrix)
  • uploaded documents (for example, ERG and PGF)

Prison only

  • refer nominal from prison
  • link to DPS profiles
  • check categorisation and re-categorisation

Probation only

  • refer nominal from community

Remove access to pathfinder

The user account is deactivated but can still be viewed by a Pathfinder admin. The user cannot login to Pathfinder anymore.

Group Owner

The user manages a group. They can ask to add and remove Pathfinder users.

Controls Intervention Matrix approver

The user can approve a Controls Intervention Matrix.
